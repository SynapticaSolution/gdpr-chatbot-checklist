# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately rather than opening a public issue.

- **Email**: brain@synaptica-solution.com
- Include a description of the vulnerability, steps to reproduce, and potential impact.

We'll acknowledge your report as soon as possible and work on a fix or mitigation. Please give us reasonable time to address the issue before any public disclosure.

## Supported Versions

Only the latest version on the `main` branch is actively maintained and supported with security fixes.
